# Financial Tracking, Analysis and Visualization for Small Businesses

## Overview
Financial Tracking, Analysis and Visualization for Small Businesses is a data-driven project developed to help small and micro businesses manage their financial records efficiently. The system automates financial data processing, provides interactive visualizations, and applies machine learning techniques for business insights and forecasting.

This project reduces manual bookkeeping efforts and enables business owners to make better financial decisions using analytics.

## Features
- Upload financial datasets in Excel format
- Automated data cleaning and preprocessing
- Revenue and expense analysis
- Monthly revenue visualization
- Category-wise sales analysis
- Customer Lifetime Value (CLV) prediction
- Top customer identification
- Sales forecasting using machine learning
- Interactive dashboard using Gradio

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Gradio
- Jupyter Notebook

## Machine Learning Models
This project uses:
- **Linear Regression** → Sales forecasting
- **Gradient Boosting Regressor** → Customer Lifetime Value (CLV) prediction
